Interpretating Search Results

  1. ‘User Search History’ in ‘Search Results and History’ tab shows search results with ‘Search Title’. User can visualize the results its analysis by clicking the corresponding ‘Search Title’ row.
  2. User can download results by clicking ‘Results Download’ button option provided on top right. The results will be downloaded as a *.zip file.

    3. Downloadable Result Files

    • Downloaded result (*.zip) file contains multiple following sub folders and files.
      • - A number of sub folders corresponding to different peptides in the protein and each folder contains given below set of information:
        a) Subfolders containing rate of oxidation for individual replicate.
        b) Dose response plots of rate of oxidation.
        c) Mean_R1_R2_R3.xls file contains the mean values of (1 - Rate of oxidation) for each modified residue reported in that peptide.
      - PeptideInfo.xls file which includes information regarding the peptide sequence and their corresponding peptide folder.
      - PF_SASA_tab.xls a table that contains log Protection factor of each modified residue.
      - SASAmain.png a graphical representation of the protection factor data with SASA values on x-axis and log (PF) on y-axis.
      - Modified.pdb a pdb file incorporated with log(PF) values.
      - ResultsBridge.xlsx an excel file that contains centrality scores of the residues using Bridge algorithm.
      - frustratometeR_Results include images that quantifies the degree of local frustration in a given protein. User also get tables with the calculated Frustration indexes and scripts to locally explore and manipulate the structures.
  3. User can visualize the Search query Parameters by clicking the ‘XFMS Search Query Parameters’.
  4. User can visualize XFMS results by clicking the ‘Results’ button in ‘Protection Factor’.
    • ‘Detailed Protection Factor Results’ shows protein information, solvent accessibility graph, and the protection factor table shows the anomalous residues and their solvent accessibility.
  5. Protein Information includes the name of the protein, organism name, gene name and number of amino acids in the protein followed by its sequence.
  6. The graph contains SASA values on x-axis and log (PF) on y-axis.
  7. ‘Residue Based Solvent Accessible Surface Area & Log (Protection Factor)’ shows the details of the anomalous residues. The table shows Residue position, residue name their SASA value and log (PF) scores.
  8. In ‘Betweeness Centrality’ tab by clicking the ‘Results’ button, user can visualize the centrality scores of important residues identified by using Bridge Algorithm.
    • ‘Detailed Betweenness Centrality Results’ shows protein information as well as important hydrogen bonded residues and their degree centrality and betweenness centrality scores.
    • Protein Information includes the name of the protein, organism name, gene name and number of amino acids in the protein followed by its sequence.
    • The table shows the name and position of important hydrogen bonded residues and their degree centrality and betweenness centrality scores.
  9. In ‘Degree of Frustration’ tab by clicking the ‘Results’ button, user can visualize the locally frustrated contacts calculated by analyzing energy distribution using frustratometeR algorithm.
    • ‘Detailed Frustration Results’ include degree of frustration in each class, contacts density in sequence space and frustration contact map.
    • Protein Information includes the name of the protein, organism name, gene name and number of amino acids in the protein followed by its sequence.
    • ‘Fraction Of Contacts In Each Frustration Class’ represents the frustration density. The x-axis represents the residue position in sequence and y-axis represents the density of frustration. Green color represents minimally frustrated and red represents highly frustrated.
    • ‘Density Of Contacts In Sequence Space’ shows the frustration in sequence. The x-axis represents the residue position in sequence and y-axis represents the local frustration density.
    • ‘Frustration Contact Map’ represents the local frustration in contacts. The frustration index measures how favorable a particular contact is relative to the set of all possible contacts in that location. Green color represents minimally frustrated and red represents highly frustrated.
